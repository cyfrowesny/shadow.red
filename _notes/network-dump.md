## Commands for displaying information about network interfaces:


### Option #1:

```bash
ip addr
```

### Option #2

```bash
ifconfig
```

### More detail information about network cards
This command displays information about network cards/adapter (ID, Interface, Driver, Chipset):

```bash
sudo airmon-ng
```
